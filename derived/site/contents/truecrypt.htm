About TrueCrypt 6.3a

TrueCrypt enables the user to create and maintain an on-the-fly-encrypted volume (data storage device). The term "On-the-fly" refers to the ability of TrueCrypt to automatically encrypt data before it is saved or decrypt data just before it is loaded into memory without user intervention. Data stored on an encrypted volume can only be read if the correct password or encryption key is used.

Be sure to read the TrueCrypt User Guide (PDF) before installing or the online beginner's tutorial.

Features of TrueCrypt 6.3a

  • Can encrypt the entire file system i.e. file names, folder names, contents of every file and free space.
  • Unencrypted data is never stored on any storage device.
  • TrueCrypt volumes can be stored on recodable CDs, DVDs, and other read-only media.
  • TrueCrypt can encrypt floppy disks, ZIP disks, USB hard disks, USB memory sticks and other types of storage devices.
