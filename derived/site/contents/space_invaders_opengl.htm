About Space Invaders OpenGL 0.6.1

Space Invaders OpenGL is Windows-based version of the classic arcade game "Space Invaders". It requires an OpenGL compatible 3D accelerator video card and OpenGL drivers.

If you experience an error message about msvcr70.dll, you need to download this zip file, then extract and install msvcr70.dll.

  • For Windows 2000/XP: install msvcr70.dll in the "windows/system32" directory.
  • For Windows 98/ME: install msvcr70.dll in the "windows/system" directory.
