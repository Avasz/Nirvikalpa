About wxDownload Fast 0.6.0

TrueDownloader is a open source download manager for Windows 2000/XP/Vista to download files via http or ftp. It has several useful features including : download resuming (pause and restart downloads), download scheduling, Proxy support and ZIP previewing.

Features of wxDownload Fast 0.6.0

  • support for HTTP and FTP protocols including downloading from HTTP and FTP servers which require a password.
  • Integrates with Mozilla Firefox via the FlashGot extension
  • Metalink support
