About winMd5Sum 1.0.1.55

WinMd5Sum is a utility to calculate the MD5 checksum of a file.

Essentially, a MD5 program like WinMd5Sum scans a file and generates a 32 digit hexadecimal number. If the file is changed, even by one byte, the MD5 algorithm will calculate a new checksum. Hence, end users can check that downloaded files are not corrupted by comparing the MD5 checksum of your downloaded file with that posted on the software's website. If these MD5sums are the same, then your file is the same as the file on the website.
