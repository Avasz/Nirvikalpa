About Spirex 2.25

Spirex is a Windows (98 and later) screensaver based on the geometry of a Spirograph and uses continuously varying wheel sizes to draw complex 3D shapes.

For optimum performance, your video card should support OpenGL hardware acceleration. (Download/view Spirex movie)

Features of Spirex 2.25

3D extension of a Spirograph.
Texture mapping of 3D solids.
You can use your own images for the texture mapping.
Anti-aliased lines in 2D modes.
User can enable random switching among presets.
Can save user presets.
Optional battery saver mode (for Windows only).
Most controls have tool tips (rest the mouse over them for a few seconds to see the tip if available).
