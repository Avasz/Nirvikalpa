About Tux Paint 0.9.21

Tux Paint is a free drawing program for Windows 2000/XP/Vista designed for children (age 3 and up). It has a simple, easy-to-use interface, fun sound effects and a cartoon mascot who helps guide children as they use the program. It provides a blank canvas and a variety of drawing tools to help your child be creative.

Features of Tux Paint 0.9.21

  • Fun sound effects are played when tools are selected and used.
  • Paint brush - Various brushes available and can add more brushes.
  • Rubber stamp - Various stamp images available and Can add more stamps and create categories. Stamps can have text (names, facts, etc.) assigned to them.
  • Line tool - Uses Paint Brush's brushes. "Rubberband" shows where the line will be drawn as you move the mouse.
  • Shape tool - Draw various filled and unfilled shapes. Rotate shapes (can be disabled if necessary).
  • Text tool - With various fonts and sizes and can add your own fonts.
  • "Magic" (special effects) tools includes :
    • Grass - easily draw grass. Don't forget the dirt!
    • Bricks - draw a brick wall
    • Rainbow brush - draw using a brush that paints in all colours of the rainbow.
    • Sparkles - draw glowing yellow sparkles.
    • Mirror - flip the image horizontally.
    • Flip - flip the image vertically.
    • Blur - blur parts of the picture using the mouse.
    • Smudge - push the colours around as though smudging paint.
    • Fade - fade the colours of parts of the picture.
    • Darken - darken the colours of parts of the picture.
