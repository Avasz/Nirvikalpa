About Inkscape 0.47

Inkscape is an open source drawing tool similar to Illustrator, Freehand, and CorelDraw. It uses the W3C standard scalable vector graphics format (SVG).

Features of Inkscape 0.47

  • Supports SVG alpha blending.
  • Supports SVG XML editing.
  • Imports formats such as: EPS, Postscript, JPEG, PNG, BMP and TIFF.
  • Exports to PNG and other multiple vector-based formats.
