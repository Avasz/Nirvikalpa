About MusikCube 2.0 developer milestone 3.3.2

MusikCube 2 is a music "library" application similar to Apple iTunes. Users can organise their music by album, genre, random access etc. It has limited support for net radio.

MusikCube supports windows 2000, XP and higher.

Features of MusikCube 2.0 developer milestone 3.3.2

  • Supports mp3, ogg, flac, and other cd audio file formats via plugins.
  • Integrated cd ripper.
  • Integrated cross fader.
  • Drag and drop compatible.
  • Batch tagging.
  • Add multiple folders (local and network)
