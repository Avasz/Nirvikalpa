About Tux Typing 2 1.8.1

Tux Typing is an educational typing tutor for children which features several different types of gameplay with a variety of difficulty levels.

Features of Tux Typing 2 1.8.1

For the end user:
Tux Typing offers a simple and fun way to learn how to type with a standard QWERTY keyboard.
For parents:
Children can play and learn to type all at the same time.
For teachers:
Tux Typing is a flexible tool for use in the classroom. It is easier to group different types of exercises, words and characters into a lesson.
Includes: Comet Zap! a typing game in which Tux must protect 9 cities.
