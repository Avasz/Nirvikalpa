About FileZilla 3.3.3

FileZilla is a fast and reliable cross-platform FTP, FTPS and SFTP client for Windows 2000/XP/Vista which allows you to upload/download files from FTP (file transfer protocol) servers. It features an Explorer style interface to show the local and remote folders.

Features of FileZilla 3.3.3

  • Supports resume and transfer of large files >4GB
  • Cross-platform. Runs on Windows, Linux, *BSD, OSX and more
  • Site Manager to manage the FTP sites (and its associated connection details) you connect to. The Site Manager also allows grouping of related FTP sites in folders to organise them.
  • Supports FTP, FTP over SSL/TLS (FTPS) and SSH File Transfer Protocol (SFTP)
  • Drag and drop between local and remote folders.
  • Queue multiple files to be uploaded/downloaded.
  • Works with firewalls.
  • Has support for SOCKS4/5 and HTTP1.1 proxies.
  • Can communicate with FTP servers using Kerberos with GSS.
  • Displays a message log which displays the FileZilla commands and the responses from the remote server.
