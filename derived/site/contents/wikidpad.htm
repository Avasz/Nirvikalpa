About WikidPad 1.9

WikidPad is a standalone notepad like application with a wiki-style interface for storing your thoughts, ideas, todo lists, contacts, or anything else you can think of to write down. The term wiki means "quick" in Hawaiian, and wikis are all about quickly linking your information together. There are many web based wiki servers available, the most famous wiki being Wikipedia, the free encyclopedia.

wikidPad uses the same concepts as wikis to easily cross-link your information by typing in WikiWords (A WikiWord is any mixed case word typed into the editor) ; Wikidpad creates these links on the fly ; clicking on the Wikiword opening that section ; see the Wikidpad Documentation online for more examples

Features of WikidPad 1.9

  • Install WikidPad to your flash or removable drive so you can take WikidPad wherever you work
  • On the fly WikiWord linking, as you type
  • WikiWord auto-completion
  • can insert links to files by typing the full path to the file or by drag and drop the file into the Wikidpad editor/li>
  • Can export the Wiki to HTML or XML
