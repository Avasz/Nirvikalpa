About WinMerge 2.12.4

WinMerge allows you to visually compare text files. Differences from one text file can be quickly moved to the other text file.

Features of WinMerge 2.12.4

  • Visual differencing and merging of text files
  • Flexible text editor with syntax highlighting
  • Handles DOS, UNIX and MAC text file formats
  • Unicode support
  • Difference pane shows current difference in two vertical panes
  • Highlights differences inside lines
