About XAMPP Lite 1.7.2

Installing the Apache web server, MySQL and PHP and configuring them to work with each other is not a simple task. Yet this combination of software can run content management systems such as weblogs or blogs and wikis and other database driven websites.

XAMPP Lite is an easy to install Apache distribution containing MySQL and PHP for Windows (and other operating systems): just download, extract and start.

Features of XAMPP Lite 1.7.2

  • Easy to install
  • Once installed, you can try a variety of blog software such as WordPress.org and Pivot and wikis such as coWiki.

XAMPP Lite Install Notes

Note that the installation of XAMPP Lite is geared for making it easy to use in trusted environments (such as your LAN) and not in production environments (ie. to allow untrusted users to access the server over the Internet) due to security issues.

How to install :

  • Install / Download XAMPP Lite 1.7.2
  • Extract it to C:\ and it will create C:\xampplite as the installation folder.
  • Run C:\xampplite\setup_xampp.bat to complete the install.

How to run XAMPP Lite

  • Run C:\xampplite\xampp_start.exe. If you have a firewall running, it would warn you about XAMPP Lite.
  • Open your browser and point to http://localhost. You should see the XAMPP logo and a choice of languages. Select English.
  • And its running! Browse to http://localhost/security/index.php for an overview of the security status of your XAMPP installation (and what to do to better secure it).
  • To stop XAMPP Lite, run C:\xampplite\xampp_stop.exe.

And to uninstall, just delete the c:\xampplite directory (once XAMPP is not running)
