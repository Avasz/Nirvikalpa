About Coolplayer build 219

CoolPlayer is an audio player similar in function to Winamp. It can files with the following formats: MPEG audio layer 1, 2 and 3.

Audio cd playback requires a cd player plugin. Use version 1.95.

Features of Coolplayer build 219

  • Winamp input plugins support.
  • OGG Vorbis support.
  • MP3->wav converter.
  • Internet streaming.
  • Advanced Playlist editor.
  • ID3 Multitagger.
  • 8 band precision equalizer.
  • Support for skins.
