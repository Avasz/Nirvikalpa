About Programmers Notepad 2.0.10

Programmers Notepad is a text editor with extra functions for programmers and web developers. It has many features such as: a tabbed mulitple document interface and syntax highlighting for many languages including C/C++, Java, HTML, CSS, XML, Visual Basic and Perl.

Features of Programmers Notepad 2.0.10

Tabbed Multiple Document Interface (MDI).
Programmers Notepad uses tabs at the top of the program which allows you to easily switch between opened documents.
Syntax highlighting of over 20 languages
You can also customise the default colouring and fonts for each language and create your own user-scheme for syntax highlighting.
Code Folding/Outlining
This allows you to collapse and expand sections of code/html in order to focus on the parts of your document that require attention.
External tool support with output presented in Programmers Notepad
Easily run external programs such as compilers to act on files opened in Programmers Notepad. The output of such programs will be displayed within Programmers Notepad.
