About Notepad2 3.0.21

Notepad2 is a small, fast and free text editor for Windows XP and higher with syntax highlighting for HTML and other common languages. Its interface is very similiar to Windows Notepad.

Features of Notepad2 3.0.21

  • Customizable syntax highlighting: HTML, XML, CSS, JavaScript, VBScript, ASP, PHP, CSS, Perl/CGI, C/C++, C#, Java, VB, Pascal, Assembler, SQL, Python and more.
  • Drag & drop text editing inside and outside Notepad2.
  • Basic regular expression search and replace.
  • Useful word, line and block editing shortcuts.
  • Rectangular selection (Alt+Mouse).
  • Brace matching, auto indent, long line marker, zoom functions.
  • Support for Unicode, UTF-8, Unix and Mac text files.
  • Open shell links.
