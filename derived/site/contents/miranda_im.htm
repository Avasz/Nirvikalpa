About Miranda IM 0.8.27

Miranda IM is a multi-protocol instant messenger (IM) client for Windows (Windows 2000, XP or Vista) with support for multiple chat protocols. Supported chat protocols include Facebook, AIM (AOL Instant Messenger), ICQ, IRC (Internet Relay Chat), Jabber, Google Talk, MSN, and Yahoo!. Miranda IM users can log into multiple accounts on multiple IM networks simultaneously. This means that you can be chatting with friends on AOL Instant Messenger, talking to a friend on Yahoo! Messenger, and sitting in an IRC channel all at the same time.

Additional functionality, themes and support for other chat protocols can be added to Miranda IM via plugins at addons.miranda-im.org/.
