About Sumatra PDF 1.1

Sumatra PDF is a slim, free, open-source viewer of Adobe Acrobat® files, also known as Portable Document Format (PDF) files. It is a single executable file so it is portable (it can be run from removable flash drives without installing in Windows).

You can create PDFs from any Windows application wih PDFCreator
