About Rainlendar 0.22.1

Rainlendar is a customisable calendar that resides on your desktop and shows the days of the current month. You can also add events to the calendar as well as maintain a to-do list which can also be displayed on the desktop.

The calendar can use different skins to fit your taste (and theme). You can use different fonts and colors, select the background image or even make the calendar's background transparent.

Rainlendar is developed and tested for Windows XP. It should work on Win98/ME, even though they are not officially supported.

Features of Rainlendar 0.22.1

  • Supports Windows transparency (only for Windows 2000/XP).
  • Shows an alarm when an event is due.
  • Read/Write events and tasks from iCalendar (.ics) files. This feature can be used to show Mozilla Sunbird's events in Rainlendar.
  • Wide variety of skins available.
  • Displays appointments and tasks from Outlook.
  • Synchronizes events between several clients via the Rainlendar server (included in the Rainlendar folder on the CD).
