About Run With Parameters 1.5.2

Run With Parameters is a utility which allows running any application with a parameter. Instead of going to the Command Prompt, and navigating to the folder where the application is, users can simply right-click an application and enter the parameters in the pop-up window.

Features of Run With Parameters 1.5.2

  • integrates into Windows Explorer
  • remembers last parameters per application
  • supports EXE, BAT, CMD, COM, SCR, AHK, BTM
