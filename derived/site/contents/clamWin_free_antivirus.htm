About ClamWin Free AntiVirus 0.96.1

ClamWin Free Antivirus is a Free Antivirus program for Microsoft Windows 98/Me/2000/XP/2003 based on the Clam AntiVirus engine. As a standalone virus scanner, ClamWin Free Antivirus integrates with Windows Explorer ; you can right click on a file or folder to scan for viruses.

By itself, ClamWin does not include an on-access real-time scanner, you need to manually scan a file in order to detect a virus.

Features of ClamWin Free AntiVirus 0.96.1

  • ability to schedule scanning of entire drives and/or folders
  • Automatic Virus Definition Updates - the ClamAV team updates Virus definitions on a regular basis
  • Context Menu Integration in Microsoft Windows Explorer.
  • Can be used as an add-in to Microsoft Outlook.
  • Standalone Virus Scanner: does not include a real-time memory resident scanner.
