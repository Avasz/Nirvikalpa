Hardware/Software Requirements

This collection of open source software is intended for use on the following versions of Windows速: Windows XP/2000/Me/98SE. Some software can only run on Windows XP/2000. The software on the CD may or may not run on Windows 95, Windows 98 (first edition) or Windows NT. The software will not run on Windows 3.1.

The software requirements for each program/application varies but if your Windows PC is already running a program that falls in one of the aforementioned categories, it can probably run the programs listed under that category.

Your computer should have these minimum system requirements:

  • Windows速 Vista/XP/2000/Me/98SE.
  • A 300MHz or faster Intel or AMD Athlon processor/CPU.
  • 128MB or more of RAM.
  • Hard drive space: 1.5GB (for installing all of the applications) plus additional space (as needed) for data.
  • Video: 32MB (or higher) video card which supports Direct X 8.1 OR Direct X 9 and has OpenGl support (for the games).
  • Sound: Direct X compatible sound card.
  • A CD-ROM OR DVD-ROM drive, keyboard and mouse.
  • A web browser (to view the descriptions on this CD). See the How to use the CD section for browser recommendations.
